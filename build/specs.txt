User can:
	View their profiles through URI/users/:id
	View their Reimbursements through URI/reimbursements/author/userId/:userId
	Post new reimbursements through URI/reimbursements
Managers can:
	View all users through URI/users
	View any user by id through URI/users/:id
	View all reimbursements by status through URI/reimbursements/status/:statusId
	View all reimbursements by user through URI/reimbursements/author/userId/:userId
	Update reimbursements through URI/reimbursements
Admins can:
	Update users through URI/users
